# Jackpot — GitHub Pages PWA

Gói cập nhật để upload đè lên repository GitHub Pages hiện tại.

## Cách cập nhật

1. Giải nén ZIP.
2. Upload toàn bộ file bên trong vào thư mục gốc repository.
3. Chọn ghi đè các file trùng tên.
4. Chờ GitHub Pages triển khai, sau đó đóng hẳn app/web và mở lại.

## Phiên bản cache

`jackpot-shell-v2026.06.28-04`

Gói giữ nguyên mã web nguồn, bổ sung PWA, noindex và icon 🎰.
